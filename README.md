# Live Piano
Play piano with your friends in real time.

Live Piano is a piano emulator that works in real time between two or
more browsers. It uses [Tone.js](https://tonejs.github.io) for the sound, and 
[Socket.io](https://socket.io) for the real time connection.

## Install
1. Clone this repo:
```sh
$ git clone https://github.com/Ismaelleon/live-piano
```

2. Install the dependencies:
```sh
$ cd live-piano
$ pnpm i # or "npm i" if you're not using pnpm
```

3. Start the web app:
```sh
$ pnpm run dev # or "npm run dev" if you're not using pnpm
```
